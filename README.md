  三、新生端功能设计

  1. 兴趣画像测评

  - 快速问卷：10-15 道趣味问题（兴趣爱好、性格特点、时间偏好）
  - 标签生成：自动生成用户标签（如"文艺青年"、"运动达人"、"技术控"）
  - 可视化呈现：雷达图展示兴趣维度分布

  2. 智能匹配推荐

  - 匹配算法：基于兴趣标签 + 社团属性 + 时间可用性进行匹配
  - 推荐结果：
    - 最佳匹配 TOP 5（匹配度 > 80%）
    - 可能感兴趣（匹配度 60%-80%）
    - 探索更多（匹配度 < 60%）
  - 匹配理由：展示"为什么推荐这个社团给你"

  3. 一键报名

  - 统一表单：填写一次个人信息，批量报名多个社团
  - 进度追踪：实时查看报名状态（待审核/已通过/面试安排）
  - 消息提醒：面试通知、录取结果推送

  4. 招新日历

  - 时间线展示：各社团宣讲会、面试时间一目了然
  - 日程订阅：一键加入手机日历
  - 冲突提醒：面试时间冲突自动预警

  ---
  四、社团端功能设计

  1. 社团主页搭建

  - 信息展示：社团简介、活动照片、往期成果、招新要求
  - 视频介绍：支持上传招新宣传视频
  - 成员故事：学长学姐分享社团经历

  2. 报名管理

  - 报名列表：查看所有报名学生信息
  - 筛选功能：按专业、年级、兴趣标签筛选
  - 批量操作：批量通过/拒绝/发送通知

  3. 面试安排

  - 时段设置：自定义面试时间段
  - 在线预约：学生自主选择面试时段
  - 签到管理：面试现场扫码签到
